# 다관점 스코어링 시스템 v2 — 데이터 레이어 + 5개 프레임워크 + 밸류에이션 엔진

> v1(단도투자 구현명세서)을 확장한다. **v1 공통 규칙을 그대로 상속**한다: 점수 1.0~5.0, 업종 중앙값 대비 백분위(v1 §2), 근거불충분 = 2.5 상한 + 플래그, 게이트 미통과 시 WATCH 강등, 결론은 정반합.
> v2의 핵심 추가: ① 전수 영속 데이터 레이어, ② 5개 투자 관점 병렬 스코어링, ③ 자체 밸류에이션 엔진(목표가·진입가).
> 작성 기준일: 2026-06. 정책·가격·도구 정보는 변동하므로 해당 항목의 검증일을 주기적으로 갱신할 것.

---

## §0. 목적·범위

v1은 "RSI<30 → 단도 스코어링 → 신호"의 일일 EOD 파이프라인이었다. v2는 이를 **다관점 분석을 지탱하는 영속 플랫폼**으로 확장한다.

- **데이터**: 매 실행 재수집·폐기(v1) → **분기 전수 적재 + 일일 델타 보강**(영속)
- **관점**: 단도 단일 → **5개 프레임워크 병렬**(단도·LTGG·아웃사이더·마법공식·버핏/멍거)
- **밸류에이션**: 멀티플 중심(v1 C섹션) → **자체 엔진(EPV·역DCF·시나리오 DCF)으로 목표가·진입가 산출**
- **RSI<30**: v2에서도 매수 신호가 아니라 **"지금 들여다볼 종목"을 고르는 진입 타이밍 오버레이**로만 사용

**원칙(불변)**: 모든 정성 분석은 공식 출처(DART 등) 근거 강제, 사실/유추/추정/불명 구분, 최종 매수·매도 판단은 사람이 수행.

## §1. 데이터 레이어 아키텍처

```
L-Q  (분기 전수 적재)   → 정기공시 마감 다음 주, 전 종목 정기보고서 XBRL 수집·정규화
L-Δ  (일일 온디맨드)    → RSI<30 종목의 최신 정정·수시공시 보강
L-Daily (일일)         → RSI 계산 → RSI<30 필터 → DB 인출 → 단도 스코어링 → 신호 (v1 유지)
L-Multi (배치)         → 전수 DB 위에서 5개 프레임워크 + 밸류에이션 산출
```

- **L-Q 백본(방법 B)**: 전 종목·동일 기준일 데이터 → 횡단면·백분위·랭킹의 토대(특히 마법공식·업종 상대평가에 필수)
- **L-Δ 델타(방법 A)**: 분기 배치 사이의 신선도 갭과 수시공시(자사주·내부자·공급계약)를 메움
- 두 방법은 배타적이지 않다: **B를 백본, A를 델타**로 결합 = "전수 분기 DB + RSI 진입 오버레이"

## §2. 3계층 데이터 모델 (정합성의 핵심)

관점마다 원천 재무를 **중복 적재하지 않는다**. 같은 ROIC가 관점별로 제각각 저장되면 정합성이 깨진다. 대신 3계층으로 분리한다.

1. **정규화 원천 (SSOT)** — `financials_normalized`: XBRL에서 파싱한 원자적 재무를 **한 벌만** 적재(단일 진실원천). 연결/별도, NCI/지배주주 귀속 구분 유지.
2. **공통 파생지표 풀** — `metrics_derived`: ROIC·FCF·EV/EBIT·이자보상배율·NCAV·마진·주당지표 등을 **한 번 계산해 재사용**. 다수 지표를 여러 관점이 공유한다.
3. **관점별 스코어링 규칙** — `scores_multiframework`: 각 관점은 같은 지표 풀에서 **무엇을 고르고 어떻게 가중·임계·랭킹하는가**만 다르다.

→ 새 관점 추가 = 규칙(③)만 추가. 원천(①)·지표(②)는 불변.

## §3. 공통 파생지표 풀 (`metrics_derived`)

여러 관점이 공유하는 핵심 지표. 모두 `financials_normalized` + 가격(KRX)에서 산출.

| 지표 | 산식 | 주 사용 관점 |
|---|---|---|
| ROIC | NOPAT / 투하자본 (NOPAT=영업이익×(1−유효세율), 투하자본=자기자본+총차입금−현금성자산) | 단도·LTGG·버핏 |
| ROE | 지배주주순이익 / 지배주주자본 | 버핏 |
| FCF | 영업현금흐름 − CAPEX | 단도·아웃사이더·버핏 |
| EV/EBIT | (시총+순차입금) / EBIT | 단도·마법공식·밸류에이션 |
| Earnings Yield | EBIT / EV | 마법공식 |
| ROC(Greenblatt) | EBIT / (순운전자본 + 순고정자산) | 마법공식 |
| 매출 CAGR | 3·5년 매출 성장률 | LTGG |
| 마진 추세 | 영업이익률·총이익률의 회귀 기울기 | LTGG·버핏 |
| 주당가치 추세 | 주당 FCF·BVPS 성장 + 발행주식수 추세 | 아웃사이더 |
| 자사주 소각률 | 소각 자사주 / (취득 자사주) | 아웃사이더(한국 보정) |
| 순현금/NCAV | (현금성−차입금)/시총, (유동자산−총부채)/시총 | 단도·밸류에이션 |

**주의(자산가치 ≠ 실현가능가치)**: 순현금·NCAV·자회사 지분은 **소수주주가 실제 실현 가능한 경우에만** 하방으로 인정한다(지주 구조·터널링 사례는 할인). 책상 위 장부가치를 그대로 하방으로 쓰지 않는다.

## §4. 5개 프레임워크 스코어링

각 관점은 독립적으로 1.0~5.0 하위점수 → 가중총점 → 게이트 → 5단계 결론(적극 매수/편입 · 분할 · 관심 · 보류 · 제외)을 산출한다. **관점은 서로 다른 질문을 던지므로 단일 점수로 합산하지 않는다(§5).**

### §4.1 단도투자 (Pabrai) — *하방 우선 렌즈*
v1 명세서의 A~G를 그대로 사용(하방보호 25 / 사업질 20 / 밸류 20 / 밸류트랩 15 / 촉매 10 / 경영진 10, 게이트 A·D ≥ 3.0, BUY ≥ 4.0). v2에서는 5개 렌즈 중 "안 잃는 것"을 담당.

### §4.2 LTGG (Baillie Gifford) — *장기 성장 렌즈*
소수의 예외적 성장기업을 매우 길게 보유, 소수 승자가 비대칭 수익을 낸다는 철학. 하방 게이트 대신 **성장 활주로**가 관문.

| 하위 | 평가 | 점수화 | 가중 |
|---|---|---|---|
| L1. 성장 활주로 | 5년 내 매출 2배 가능성(매출 CAGR·TAM 침투율) | 성장 백분위 | 25% |
| L2. 확장되는 해자·마진 | 총이익률 추세, 규모의 경제·네트워크 효과 | 추세+정성 | 20% |
| L3. 재투자 복리(ROIC) | 고ROIC 재투자 활주로(이익을 높은 수익률로 재투자) | ROIC 수준·추세 | 20% |
| L4. 문화·장기지향 경영진 | 창업자 주도, 내부지분, R&D 강도, 장기 자본배분 | 정성(근거강제) | 15% |
| L5. 비대칭 성장 옵션 | 시장이 터미널 가치를 과소평가하는 이유, 옵셔널리티 | 정성 | 20% |

`LTGG = 0.25·L1 + 0.20·L2 + 0.20·L3 + 0.15·L4 + 0.20·L5`
**게이트**: L1 ≥ 3.0(성장 활주로 없으면 LTGG 대상 아님). 품질 플로어: 자본잠식·계속기업 불확실 종목 제외.
**프레임워크-대상 불일치 주의**: LTGG는 B2C 소비·글로벌 성장주 설계 → 한국 종목·산업주 적용 시 적용범위 밖임을 점수 캐비엇으로 유지.

### §4.3 아웃사이더 (Thorndike) — *자본배분 렌즈*
CEO를 투자자로 보고, 이익이 아니라 현금흐름·주당가치 성장·기회주의적 자본배분을 평가.

| 하위 | 평가 | 점수화 | 가중 |
|---|---|---|---|
| O1. 자본배분 합리성 | 최고수익처로 자본 배치(재투자 ROIC·인수 규율, 무분별 증자·고가 M&A 감점) | 정성+정량 | 25% |
| O2. 자사주 가격규율·소각 | 저평가 시 매입, **소각률**(취득만 하고 안 태우면 무의미), 저가 발행 회피 | 정량+정성 | 25% |
| O3. 현금흐름·레버리지 규율 | FCF 창출, 차입의 신중한 활용, 분권화 신호 | 정량 | 20% |
| O4. 주당가치 성장 | 주당 FCF·BVPS 성장 + 발행주식수 감소/유지 | 정량 | 20% |
| O5. 경영진 정렬·자율 | 내부지분, 린 본사, 분권화 | 정성 | 10% |

`아웃사이더 = 0.25·O1 + 0.25·O2 + 0.20·O3 + 0.20·O4 + 0.10·O5`
**한국 보정(필수)**: 저배당·저자사주가 **아웃사이더식 규율인지, 터널링·밸류트랩 신호인지** 반드시 판별한다. 한국에서는 후자가 흔하므로 **소각률·관계자거래·지배구조**를 함께 본다. 상법 개정(v1 §7) 촉매와 연동.

### §4.4 마법공식 (Greenblatt) — *순수 퀀트 랭킹 렌즈*
전수 횡단면 랭킹. 단일 종목 확신용이 아니라 **상위 바스켓 식별용**.

| 하위 | 산식 | 점수화 | 가중 |
|---|---|---|---|
| G1. 이익수익률 | EBIT / EV | 전체 유니버스 랭킹 백분위 | 50% |
| G2. 자본수익률(ROC) | EBIT / (순운전자본 + 순고정자산) | 전체 랭킹 백분위 | 50% |

- **결합 랭킹** = rank(G1) + rank(G2) → 결합 백분위 → 1.0~5.0 매핑(상위일수록 고득점)
- **제외 규칙**: 금융·유틸리티·외국계·최근 IPO·초소형주, **영업적자(EBIT<0) 종목**
- **성격 명시**: 이건 바스켓 랭킹 도구다. 단일 종목 "BUY"가 아니라 "상위 N 관심권"으로 표시. 게이트 없음(랭킹 자체가 결과).

### §4.5 버핏·멍거 — *품질-가치 렌즈*
"훌륭한 기업을 적정가에." 내구적 해자 + 높고 지속적인 자본수익성 + 정직·유능한 경영진 + 안전마진.

| 하위 | 평가 | 점수화 | 가중 |
|---|---|---|---|
| B1. 해자 내구성 | 브랜드·가격결정력·전환비용의 다년 일관성 | 정성(근거강제) | 20% |
| B2. 자본수익성 질·지속성 | ROE/ROIC 수준·10년 안정성, 저레버리지 | 정량 | 25% |
| B3. 이익 일관성·예측가능성 | 이익·FCF·오너이익의 일관성 | 정량 | 20% |
| B4. 경영진 정직·능력 | 솔직성, 자본배분, 관계자거래 청결성 | 정성(게이트성) | 15% |
| B5. 안전마진(적정가) | 내재가치 대비 가격(§6 엔진 연동) | 정량 | 20% |

`버핏멍거 = 0.20·B1 + 0.25·B2 + 0.20·B3 + 0.15·B4 + 0.20·B5`
**게이트**: B2 ≥ 3.0(자본수익성 없으면 wonderful company 아님). **B4 경영진 정직 = 하드게이트**: 터널링·관계자 피해 확인 시 결론 상한 Watchlist(§5 글로벌 게이트와 동일).

## §5. 교차 프레임워크 종합

**제1원칙 — 단일 합산 평균 금지.** 5개 관점은 목적·전제가 다르다(단도=하방, LTGG=성장, 아웃사이더=자본배분, 마법공식=퀀트랭킹, 버핏=품질가치). 산술 평균은 의미를 왜곡한다. 한 종목이 마법공식 5점이면서 단도 하방게이트는 탈락할 수 있고, 그 사실 자체가 정보다.

**산출 형태:**
- **다관점 대시보드**: 5개 점수 + 각 게이트/플래그를 병렬 제시
- **합치도/괴리도**: 몇 개 관점에서 동시 고득점인지, 어디서 갈리는지
- **정반합 종합 서술**: 관점 간 합의·충돌을 정·반·합으로 해석(사람이 읽는 결론)

**글로벌 거버넌스 하드게이트(전 관점 공통):** 확인된 터널링·관계자 피해·미해결 지배구조 실패 → **모든 관점 결론 상한 Watchlist**. 밸류에이션·성장성과 무관하게 적용.

**한국 보정 레이어(전 관점 공통):**
- 저배당·저자사주: 규율 vs 터널링 판별, **자사주 소각률**을 핵심 지표로
- 상법 개정 촉매: v1 §7 타임라인·자사주 의무소각 연동
- 자산가치 ≠ 실현가능가치: 소수주주 실현 가능성으로 하방 인정 여부 결정

**(선택) 합성점수**: 관점별 가중 합성을 원하면 산출 가능하나 **"보조·주의" 라벨** 필수. 기본 뷰는 다관점 병렬.

## §6. 자체 밸류에이션 엔진 (목표가·진입가)

DCF 전수 자동화의 함정(WACC·영구성장 가정 민감 → "정밀해 보이는 오답")을 피하기 위해 **2단계**로 구성.

**전수 단계(가정 적은 견고한 방법):**
1. **멀티플**: EV/EBIT·PER의 자기 5년 밴드·업종 대비 위치(v1 C)
2. **자산 플로어**: 순현금·NCAV → 하방 앵커. **단, 실현가능가치만**(§3 주의)
3. **EPV(Earnings Power Value)**: 정상화 EBIT×(1−유효세율) / WACC — 성장 예측 불필요
4. **역DCF**: 현재 주가가 함축하는 내재 성장률을 역산 → "그 성장률이 보수적인가, 무리한가"로 판단. 예측 부담 없이 비대칭성 점검에 최적.

**finalists 단계(게이트 통과 소수만, 정밀):**
- **시나리오 DCF**: bear/base/bull + 확률가중 기대수익

**산출 정의:**
- **목표가 = 단일점이 아니라 레인지**(보수적 base ~ bull)
- **진입가 = 보수적 내재가치 × (1 − 안전마진)**, **자산 플로어로 하방 앵커**. 안전마진은 품질·확실성에 따라 30~50%.
- **WACC 입력 명시**: 무위험(국고채 10년) + ERP×β(가격 시계열 산출). 영구성장률 보수적. **민감도표(가정별 결과) 출력 필수.**
- **증권사 목표가**: 입력이 아니라 **교차검증·소수의견 참고용**으로만(포워드 컨센서스 스크래핑 제약과 일관).

## §7. RSI<30 진입 오버레이 연동 (일일)

- L-Multi가 전수 DB로 5개 관점 점수를 **분기/주간 배치**로 미리 산출해 DB에 적재
- 매 영업일: RSI 계산 → RSI<30 필터 → 해당 종목의 **사전 산출된 다관점 점수 + 밸류에이션을 DB에서 인출** → 알림
- 즉 RSI는 "지금 싸진 종목 중 어떤 걸 먼저 볼지" 고르는 트리거, 점수는 미리 계산된 것을 꺼내 쓰는 구조

## §10. 리스크·한계 (필수 인지)

- **프레임워크-대상 불일치**: LTGG(글로벌 성장)·아웃사이더(미국 자본배분 사례)를 한국 종목에 적용 시 적용범위 밖임을 점수 캐비엇으로 상시 유지
- **마법공식은 바스켓 도구**: 단일 종목 확신이 아니라 상위 랭킹군. 과적합·소형주 편향 주의
- **자산가치 ≠ 실현가능가치**: 소수주주 실현 가능성으로만 하방 인정
- **DCF 민감성**: 전수 자동 DCF 금지, finalists만, 민감도표 동반
- **LLM 환각**: 정성 항목은 DART 원문 grounding 강제, 최종 판단은 사람
- **데이터 신선도**: 분기 사이엔 수시공시·정정만 델타로 반영
- **거버넌스 하드게이트**: 터널링 확인 시 전 관점 상한 Watchlist
