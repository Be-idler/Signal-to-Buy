# 클로드 채팅 심층분석 프롬프트 모음

텔레그램 질의응답 봇의 정량 분석 리포트를 클로드 채팅(claude.ai)에 붙여넣어
**정성 분석을 추가한 최종 분석**을 만들기 위한 스킴별 프롬프트입니다.

## 사용법

1. 텔레그램에서 종목 분석 리포트를 받는다 (예: `삼성전자 버핏` 질의).
2. 리포트 하단 「클로드 심층분석 핸드오프」 블록의 링크에서 해당 스킴 프롬프트를 연다.
3. 클로드 채팅에 **① 프롬프트 파일 전체 → ② 텔레그램 리포트 전체** 순서로 붙여넣고 전송한다.
4. 클로드가 근거 자료를 요청하면 DART 공시 원문·사업보고서 발췌를 제공한다
   (웹검색이 켜져 있으면 클로드가 직접 검색한다).

## 주의

- 채팅은 실시간 시세·시가총액에 접근할 수 없으므로 **리포트에 포함된 기준일
  종가·시총·재무기준을 그대로 사용**하도록 설계되어 있다. 리포트가 오래됐으면
  봇에 재질의해 최신 리포트를 받아라.
- 프롬프트의 루브릭·가중치·게이트는 저장소 `dhandho/frameworks*.py` 구현과
  일치하도록 유지한다. **코드의 배점을 바꾸면 프롬프트도 함께 갱신할 것.**
- 산출물은 참고용 시나리오이며 투자 자문이 아니다. 최종 판단은 사람.
